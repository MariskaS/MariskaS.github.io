# БФ СЕГОДНЯ

проект написан с использованием:
* [less](http://lesscss.org/).
* [bootstrap](http://getbootstrap.com/).
* [kendo ui](http://www.telerik.com/kendo-ui).

**программисту** - kendo ui использован только для облегчения процесса верстки,
при поднятии сайта необходимо исключить его исользование, т.к нам нужна поисковая оптимизация

### пункты "о нас" и "контакты"

### пункт "Как помочь"

### пункт "Стать подопечным"

## меню - левое боковое ``` .sidebar-left ```

### Раздел "Благодарности"
содержит:

1. общий список благодарностей
  * не содержит миниатюр изображений, вместо них, в зависимости от типа оказанной помощи для фонда, проставляется иконка:
    * финансовая помощь - ``` <i class="g-icon-20 i-ruble"></i> ``` иконка рубля
    * материальная помощь - ``` <i class="g-icon-20 i-shopping-cart"></i> ``` иконка карзины
    * покупка мед оборудования или лекарств - ``` <i class="g-icon-20 i-stethoscope"></i> ``` иконка стетоскопа
  * в случае если к благодарности прикладывается фото - текст благодарности помещается в ссылку и по клику открываем окно с фотками

2. вывод количества публикаций (благодарностей)

3. пейджинг - выводить не менее 15и публикаций

3. блок "Финансовая помощь"
